			2.9).React简介
				将功能分解化，让开发像搭积木一样，快速，可维护。
				特点：
					1.React可以只作为视图，在MVC中使用； 作为UI
					2.虚拟DOM， 一千，我们更新视图，需要先清空DOM容器中的内容，额然后将最新的DOM和数据追加到容器中，现在的React将这个操作放进了内存，React认为内存的操作远远比视图全部更新来的高效。React将视图变化，房间内存进行比较，计算出最小更新的视图，然后将差异的部分进行更新，这就是高效的原因。
					3.单向数据流： React实现了单向数据流，相对于传统的数据绑定而言，React更加灵活啊，便捷。
					4.来个例子：
						引入：
						<script type="text/javascript" src="react.js"></script>
						<script type="text/javascript" src="JSXTransformer.js"></script>
						HTMl：
						<div id="example"></div>
						jsx:
						var HelloMessage = React.createClass({
							render: function(){
								return <h1>hello {this.props.name}</h1>
							}
						});
						React.render(<HelloMessage name="React"/>, document.getElementById('example'));
						<script type="text/jsx">
						</script>


			2.10).React Native
				一次学习，多个平台编写代码，
				只要掌握了React 和JSX，补充IOS/Android/Web只是，就能开发Native应用
				上例子：
					创建一个demo名的项目
					react-native iknit demo
					使用Xcode打开项目，修改index.ios.js:
					'use strict';
					var React = require('react-native');
					var {
						AppRegistry,
						StyleSheet,
						Text,
						View,
						Image,
					} = React;
					var demo = React.createClass({
						render : function(){
							return (
								<View style={styles.container}>
									<Text style={styles.welcome}>欢迎来到React Native世界</Text>
									<Image style{{width:50,height:50,resizeMode:Image.resizeMode.contain}}
									 source={{uri:'https//facebook.github.io/react-native/img/header_logo.png'}}>
									</Image>
								</View>
							)
						}
					})
					var styles = StyleSheet.create({
						container : {
							flex : 1,
							justifyContent : 'center',
							alignItems : 'center',
							backgroundColor : "#f50"
						},
						welcome : {
							fontSize:20,
							color:'#fff'
						}
					})
					AppRegistry.registerComponent('demo',() => demo);
					接着再Xcode中启动项目

					分析代码：
						使用require引入react-native模块，定义了AppRegistrey、StyleSheet、Text、View和Image这5个对象，
						其实是React Native的3个组件Text/View/Image和2个API
						上面的定义也可以是这样的：
							var AppRegistry = React.AppRegistry;
							var StyleSheet = React.StyleSheet;
